# PROJETO FINAL

##  Sobre o Projeto

O  projeto em sitem como principal objetivo o desenvolvimento de um sistema integrado entre um **PDV (Ponto de Venda)** e um **sistema de delivery**, proporcionando maior praticidade tanto para os clientes quanto para os funcionários da empresa.

O sistema permitirá que o cliente (consumidor final) visualize os produtos disponíveis para venda de forma online, podendo realizar pedidos diretamente pela plataforma.

---

##  Funcionalidades do Cliente

O cliente poderá:

- Visualizar os produtos disponíveis;
- Escolher a quantidade desejada;
- Selecionar a forma de recebimento:
  - Retirada no local;
  - Entrega delivery;
- Escolher a forma de pagamento;
- Finalizar pedidos de maneira prática e rápida.

---

##  Integração com o PDV

O sistema de delivery será totalmente integrado ao PDV, que funcionará como o sistema de retaguarda da aplicação.

Através do PDV será possível:

- Receber os pedidos realizados;
- Aceitar ou rejeitar pedidos;
- Atualizar o status dos pedidos;
- Finalizar vendas;
- Gerenciar produtos e operações do sistema.

---

##  Controle de Permissões

O sistema contará com diferentes níveis de acesso para garantir maior segurança nas operações.

### Funcionário
O funcionário terá acesso limitado ao sistema, podendo realizar apenas operações permitidas pelo seu perfil.

Entre as restrições definidas:

- Não poderá cancelar vendas finalizadas sem autorização;
- Para realizar cancelamentos será necessária a senha de um supervisor.

### Supervisor
O supervisor possuirá o perfil de acesso com permissão de **SUPERVISOR**, sendo responsável por:

- Autorizar cancelamentos de vendas;
- Validar operações críticas do sistema;
- Garantir maior controle e segurança nas transações.

---

## Objetivo do Projeto

O principal objetivo é criar um sistema moderno, funcional e integrado, oferecendo:

- Maior agilidade no atendimento;
- Melhor controle de vendas;
- Facilidade no gerenciamento de pedidos;
- Segurança nas operações do sistema;
- Melhor experiência para o cliente final.
